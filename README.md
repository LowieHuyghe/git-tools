# Git Tools

Tools to make working with Git even easier.


## Installation

1. Clone the project:

 ```bash
git clone git@github.com:LowieHuyghe/git-tools.git
```
2. Move into the new directory:

 ```bash
cd git-tools
```
3. Install the requirements:

 ```bash
npm install
```
4. Compile:

 ```bash
npm run compile
```


## Run

 ```bash
node dist/index.js
```
